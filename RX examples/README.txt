Задание 1.
Создать класс со статическим методом, создающим Observable используя метод create. Observable эмитет сигнал с числом,
 затем завершается. Число рандомное, observable hot.
Во ViewController, в методе viewDidLoad, реализовать подписку на созданный Observable, используя оператор subscribe.
 Реализовать только те замыкания (всего их 4), которые могут быть вызваны. Замыкания для сигналов, которые никогда не будут
 прокинуты текущим Observable, не должны быть реализованы.
